now i shall plug all my fav stuff that i use if u wanna be like me:

i use arch for my desktop + cachyos (same thing anyways bweh) for my laptop. my phone runs grapheneos & u should use it too! desktopwise i use hyprland with waybar on both my laptop and desktop. for my most oft used software, i use surf, claws mail, heroic games launcher, and qownnotes (what i use to write blog posts). i use kitty with fish shell. for programming and making this website i use vscodium with vim keybinds; my general text editor is neovim.

if i have anything to really recommend specifically, it'd have to be these. give them a try, i love them!

  • - nnn which is a minimalist file manager. i used to use midnight commander which is also great but nnn is so much quicker to use imo and it has all you need.
  • - foliate which is imo the goat of linux e-readers
